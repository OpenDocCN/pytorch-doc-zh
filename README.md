## 下载

### Docker

```
docker pull apachecn0/pytorch-doc-zh
docker run -tid -p <port>:80 apachecn0/pytorch-doc-zh
# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g pytorch-doc-zh
pytorch-doc-zh <port>
# 访问 http://localhost:{port} 查看文档
```
